Update PRD: Spesifikasi Detail Task (Modal)
1. Informasi Header & Meta

Judul Tugas: Nama kegiatan (Contoh: Recruitment Banjarmasin 13/4/26).

Lokasi List: Menampilkan informasi di kolom mana tugas berada dan di tim mana (Contoh: di list Data Pemanggilan Interview (H-1) dalam SDM - HR Recruitment).

Deadline: Menampilkan tanggal jatuh tempo (Contoh: Apr 13).

Akses: Pengaturan siapa yang bisa melihat atau mengedit tugas ini.

2. Panel Kolaborasi (Sidebar/Top Section)

Anggota (Members): Menampilkan avatar pengguna yang ditugaskan ke task tersebut.

Label: Sistem tagging warna untuk klasifikasi (Contoh: Tim Alpha, Tim Bravo, SDM Banjarmasin, Recruitment).

3. Konten & Dokumentasi

Catatan (Notes/Description): * Mendukung teks panjang (Rich Text).

Digunakan untuk daftar kontak (Nama & Nomor WA) atau informasi krusial lainnya.

Fitur "Tampilkan Lebih Banyak" untuk menyembunyikan detail yang terlalu panjang agar UI tetap bersih.

Lampiran (Attachments): * Area unggah file (PDF, Gambar, Excel).

Anggota dapat mengunggah dokumen pendukung (Contoh: Formasi Karyawan, Database).

4. Komentar & Aktivitas (Log & Chat)

Bagian ini berfungsi sebagai audit log sekaligus media diskusi.

Aktivitas Otomatis: Mencatat kapan tugas dibuat, dipindah kolom, atau diubah detailnya.

Komentar User: * Input pesan teks.

Fitur "Balas" (Reply) untuk membalas komentar spesifik (threaded conversation).

Timestamp otomatis (Contoh: Yesterday, 05:23 PM).

Data Diri/Template: Kemampuan untuk menyalin/tempel format teks panjang (seperti form biodata karyawan) ke dalam kolom komentar atau catatan.
